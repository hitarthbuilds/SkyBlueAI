# SkyBlueAI - Football Intelligence Suite

## Summary
SkyBlueAI is an on-prem, real-time football intelligence platform that merges video, event data, and ML-driven insights to cut analyst workload and improve match preparation quality. Designed for professional clubs, with Manchester City as the primary use case.

## Core Value
- Automate 60-80% of manual match analysis
- Generate actionable opponent insights within minutes
- Live tactical intelligence during matches
- Reduce injury risk through proactive load monitoring

## Modules
- Opponent Weakness Scanner
- Player Performance Predictor
- Injury Risk Engine
- Set-Piece Generator
- Tactical AI Coach
- Pro Dashboard with exports

## Live Capabilities
- Real-time event ingestion
- WebSocket live updates
- Rolling heatmaps, xThreat, pressing intensity

## Deployment
- On-prem Docker stack
- Postgres + Redis + MinIO
- Nginx + Gunicorn backend
