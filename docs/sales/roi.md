# ROI Model (Template)

## Baseline Assumptions
- Analysts spend 10-15 hours per match on manual analysis
- 1-2 analyst roles saved or reallocated
- Reduced injury incidents from proactive alerts

## Savings
- Analyst time: 50-80% reduction
- Faster prep cycles and decision-making
- Reduced external scouting overhead

## Gains
- Improved tactical adjustments
- Increased set-piece conversion rates
- Better player load management

## Notes
This model is a template. Use actual club staffing, injury history, and match volumes for an accurate ROI estimate.
